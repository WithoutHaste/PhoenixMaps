NIX File Format Definition

NIX is a text-based file format for Phoenix Maps.

File Format

Both sections are optional.
The tiles section defines the image part of the map.
The labels section defines text on top of the map. 
VERSION
[version]
			
TILES
[tiles]

LABELS
[labels]

[version] Section

The name of the tile set.

[tiles] Section

The tiles section is a csv (comma-separated values).
• No spaces
• No quote marks
• One row of the map per row of csv 
grass,floor-nw-wall,floor-n-wall,floor-ne-wall
grass,floor-w-wall-door,floor,floor-e-wall

[labels] Section

The labels section is a csv (comma-separated values).
• Surround each label with double-quotes
• Do not use double-quotes within a label
• Each unit of label corresponds to one tile on the map

Example of displaying map coordinates: 
"A1","A2","A3","A4"
"B1","B2","B3","B4"

Example of one label and a lot of blanks: 
,,,
,"B2",,
